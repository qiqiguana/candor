The API package of the Control module contains any public API consumers will have to know to be able to communicate with the a Summa Control, Client, or Service. The classes inside the API package is packed into 

    summa-control-api-<version>.jar

The public API of the Control module can be expected to be stable. At present there are no guarantees, but we strive to grant that in the future.

Implementations of actual clients and services will have to depend on the entire Control module.
