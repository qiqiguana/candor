The API package of the Search module contains any public API consumers will have to know to be able to communicate with a search engine. The classes inside the API package is packed into 

    summa-search-api-<version>.jar

The public API of the Search module can be expected to be stable. At present there are no guarantees, but we strive to grant that in the future.

Implementations of actual servers will have to depend on the entire Search module.
