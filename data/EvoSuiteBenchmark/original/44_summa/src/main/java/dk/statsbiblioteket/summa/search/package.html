Searching is the act of sending a request which delegates to one or more searchers. The results from the searches is then returned.

The SummaSearcher defines the outer interface, SearchNode defines the underlying searchers.

SearchNodeFactory - constructs SearchNodes from properties. SummaSearcherImpl - handles queueing of requests to a single SearchNode. SearchNodeAggregator - contains one or more SearchNodes that are searched in parallel or in sequence, depending on setup. SearchNodeImpl - base inplementation of SearchNode that handles threading and open/close/warmup coordination.
