Extends XML transformation functionality. A plugin is a class with only static functions. Plugins are used in indexing for extending XML transformation power. Typical plugins perform String manipulation operations, difficult to express in XSLT. Plugins is designed to run inside an XSLT and should never throw exception - including runtime exceptions.
